# Evaluation Quality Rubric

This rubric is applied to every evaluation scenario. Each category is scored on a 1-5 scale.

---

## 1. Personality Consistency

Agents maintain their defined personality traits across ticks. An agent described as "rebellious" should not suddenly become compliant without narrative justification. An agent described as "diplomatic" should not resort to insults.

| Score | Description |
|---|---|
| 5 | Every agent stays fully in character throughout the simulation |
| 4 | Minor personality drift in 1-2 agents, but core traits hold |
| 3 | Some agents lose distinctive voice by mid-simulation |
| 2 | Multiple agents converge to a generic communication style |
| 1 | Agents are indistinguishable from each other (personality collapse) |

---

## 2. Rule Awareness

Agents acknowledge and react to new rules when introduced via triggers. They should reference the rule content, adjust their behavior, and discuss the implications with other agents.

| Score | Description |
|---|---|
| 5 | All agents acknowledge the new rule and adjust behavior accordingly |
| 4 | Most agents react; 1-2 seem unaware but eventually catch up |
| 3 | About half the agents react to the rule change |
| 2 | Only 1-2 agents notice the rule; others continue as before |
| 1 | Agents completely ignore the policy trigger |

---

## 3. Social Coherence

Interactions follow established social relationships and proximity. Agents who are described as friends should interact warmly. Authority figures should be addressed with appropriate deference or challenge depending on personality. Agents should not address people they have no connection to as if they are close friends.

| Score | Description |
|---|---|
| 5 | All interactions reflect the social graph and personality dynamics |
| 4 | Most interactions are socially appropriate; minor inconsistencies |
| 3 | Some interactions feel forced or out of context |
| 2 | Agents interact randomly without regard for social structure |
| 1 | Social relationships are completely ignored |

---

## 4. Coalition Formation

Agents with aligned interests naturally group together. In conflict scenarios, factions should emerge. Mediators should be sought out. Isolated agents should either join a group or have a narrative reason for staying apart.

| Score | Description |
|---|---|
| 5 | Clear, organic coalitions form based on shared interests and personality |
| 4 | Coalitions form but with some unexpected membership |
| 3 | Loose groupings emerge but dissolve quickly |
| 2 | Agents pair up but no meaningful coalitions form |
| 1 | No coalition behavior -- all agents act as isolated individuals |

---

## 5. Narrative Emergence

Distinct story arcs emerge from agent interactions. The simulation should produce identifiable dramatic beats: conflict escalation, negotiation attempts, turning points, and resolution (or deliberate lack thereof). The overall narrative should feel like a coherent story, not random chatter.

| Score | Description |
|---|---|
| 5 | A compelling multi-threaded narrative with clear dramatic arc |
| 4 | A recognizable story with beginning, middle, and end |
| 3 | Some narrative threads but no cohesive overall arc |
| 2 | Repetitive exchanges with occasional interesting moments |
| 1 | No discernible narrative -- random or circular conversation |
